# Lab Assignment 3
# Responsive Personal Portfolio Website

1. Project Title
Responsive Personal Portfolio Website (CSS Advanced)

2. Description
This project is a one-page responsive portfolio website created using *Flexbox, CSS Grid, media queries, and animations*.
The website includes the following sections:
Header, Hero, About, Skills, Projects, Contact, and Footer.
The design is mobile-first, fully responsive, and uses relative CSS units for scalable spacing and typography.

3. Technologies Used

* HTML5
* CSS3
  * Flexbox
  * CSS Grid
  * Media Queries
  * Animations (keyframes + transitions)
  * Transforms
* Google Fonts 

4. Key Features

* Responsive layout for *mobile, tablet, and desktop*
* Flexbox-based header and hero section
* Grid-based skills and projects section
* Hover effects on buttons, links, and cards
* Simple animations (fade-in, color change, transforms)
* Contact form with Name, Email, and Message fields

5. How to View the Project

1. Open the project folder.
2. Run the file **index.html** in any browser.
3. Resize the browser window to check responsiveness.

Breakpoints tested:
* Mobile: *0–600px*
* Tablet: *600–1024px*
* Desktop: *1024px and above*
