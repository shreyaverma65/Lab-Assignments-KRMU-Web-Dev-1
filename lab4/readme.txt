*Prompt Quizzer – A JavaScript Quiz Game*

*Project Overview*

Prompt Quizzer is a console-based quiz game that:
* Uses a predefined array of questions and answers
* Takes user input using `prompt()`
* Validates answers using `toLowerCase()` and `trim()`
* Gives immediate feedback using `alert()`
* Tracks the user’s score
* Displays the final score at the end

The entire quiz runs without any HTML elements—only JavaScript executed in the browser console.

*Features*
✔ Array of quiz questions and answers
✔ Loop through questions using `for` loop
✔ User input handling and cleaning
✔ Conditional answer checking
✔ Real-time feedback
✔ Score counter
✔ Final score display



*How to Run*
1. Open any browser (Chrome, Firefox, Edge, etc.)
2. Press **F12** → Go to **Console**
3. Paste the entire JavaScript code
4. Press **Enter**
5. Answer the prompts as they appear
6. View your final score in the last alert
