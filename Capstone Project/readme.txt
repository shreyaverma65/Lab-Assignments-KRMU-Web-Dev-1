*Overview*
This project is a responsive food delivery website built using **HTML5 and CSS3**. It includes sections such as Hero Banner, Categories, Featured Restaurants, Search Bar, Reviews, App Download, and Special Offers. The UI is fully styled, mobile-friendly, and follows semantic structure.

*Features*
* Navigation bar with internal links
* Hero section with CTA
* Search bar for food & categories
* Category grid (Pizza, Burgers, Sushi, Desserts)
* Restaurant cards with ratings
* Promo codes & discount offers
* Customer reviews
* App download section
* Footer with links
* Smooth scrolling & hover animations

*Technologies Used*
* HTML5
* CSS3 (Flexbox, Grid, Media Queries)
* Google Fonts (Poppins)
* Freepik/PNG images

*Responsiveness*
The design adapts for desktop, tablet, and mobile using media queries. Grids collapse into 2-column and 1-column layouts on smaller screens.

*How to Run*
1. Open project folder
2. Run `index.html` in browser

/images (all illustration files)
